Somos Crédito es una institución que cuenta con diversas sucursales en todo el país, para lo cual
tiene la necesidad de contar con la información de la ubicación física de cada una de ellas, así
como el número de teléfono. La información se deberá poder modificar en el transcurso del
tiempo ya que las fiscalías cambian de ubicación constantemente. Se podrá dar de alta y baja a
una sucursal en el transcurso del tiempo.
